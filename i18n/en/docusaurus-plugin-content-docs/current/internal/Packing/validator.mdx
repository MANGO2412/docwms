## Validator Module

This module is used to validate the origin of an item that is going to be packaged.

### Useful View of the Module
![](/img/upload/Screenshot-2025-06-03-101136.png)

| No | Name           | Description                                                                 |
|----|----------------|-----------------------------------------------------------------------------|
| 1  | Validator Button | A button that, when pressed, opens the validator panel to validate the origin of items. |

### Validate Items

<table>
    <tr>
        <th><h3>Context</h3></th>
    </tr>
    <tr>
        <td>
            The totes or carts contain the trays where the items to be packaged are stored.

            ![logo](/img/upload/Screenshot-2025-06-26-132818.png)

            The trays are where the items are stored and have the code that is scanned to start the **validator** module.

            ![](/img/upload/Screenshot-2025-05-20-091914.png)
        </td>
    </tr>
</table>

<table>
    <tr>
        <th><h3>Steps</h3></th>
    </tr>
    <tr>
        <td>
            Select **1** to use the **Validator** tool.

            Scan the **QR code** of the **tray.** The tray information and its contents will then appear.

            ![logo](/img/upload/Screenshot-2025-06-03-114538.png)

            Scan the **QR** or **barcode** of the product. A window showing the manufacturing origin will then open.

            ![](/img/upload/Screenshot-2025-06-03-121047.png)

            Click on **select country of origin**, and a dropdown panel will open with the possible origins for this item. Select the correct one.

            ![](/img/upload/Screenshot-2025-06-03-121353.png)

            Finally, click on **continue**.

            ![](/img/upload/Screenshot-2025-06-03-121459.png)

            A **notification** will appear in the **lower right corner** of the screen indicating the **end of the validation**.

            ![](/img/upload/Screenshot-2025-06-03-104533.png)

            Afterwards, the **content labels** and **package listing labels** will be **printed**.

            ![](/img/upload/Screenshot-2025-06-03-121837.png)

            ![](/img/upload/Screenshot-2025-06-03-121907.png)
        </td>
    </tr>
</table>
