## SLAM Module

This module is used to start the packaging process.

### Useful View of the Module
![](/img/upload/Screenshot-2025-06-03-122324.png)

| No | Name         | Description                                                                 |
|----|--------------|-----------------------------------------------------------------------------|
| 1  | SLAM Button  | A button that, when pressed, opens the SLAM panel to scan the package's content label and register the packaging of the items. |

<table>
    <tr>
        <th><h3>Steps</h3></th>
    </tr>
    <tr>
        <td>
            Select **1** to use the **SLAM** tool.

            Scan the **QR code** or **barcode** of the package's content label. Package information will be displayed on the interface.

            ![](/img/upload/Screenshot-2025-06-03-110812.png)

            Now **place the box on the scale** and then **scan the box's dimension code.** The package's weight will update on the interface, and the dimensions will be saved successfully.
            
            ![](/img/upload/Screenshot-2025-06-26-134724.png)

            ![](/img/upload/Screenshot-2025-06-03-140733.png)

            Finally, click the blue "**Packaging Box**" button, and a success notification will appear.

            ![](/img/upload/Screenshot-2025-06-03-140809.png)
        </td>
    </tr>
</table>
